# PeptideSuppliers.org

Static GitHub Pages site for `peptidesuppliers.org`.

Files:

- `index.html` - homepage
- `about/index.html` - about and standards
- `faq/index.html` - visitor FAQ
- `reviews/index.html` - supplier reviews and discounts
- `contact/index.html` - contact and submission page
- `about.html`, `faq.html`, `reviews.html`, `contact.html` - redirects to clean URLs
- `styles.css` - shared site styling
- `CNAME` - custom domain for GitHub Pages
- `robots.txt` - crawler guidance
- `sitemap.xml` - page discovery for search engines
