# PeptideSuppliers.org

Static GitHub Pages site for `peptidesuppliers.org`.

Files:

- `index.html` - homepage
- `about.html` - about and standards
- `faq.html` - visitor FAQ
- `reviews.html` - supplier review templates
- `contact.html` - contact and submission page
- `styles.css` - shared site styling
- `CNAME` - custom domain for GitHub Pages
